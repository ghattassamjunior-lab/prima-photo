# Dossier Images

Placez vos images ici :

## Structure recommandée :
```
images/
├── hero/
│   └── hero-bg.jpg
├── about/
│   └── photographe.jpg
├── gallery/
│   ├── portrait1.jpg
│   ├── mariage1.jpg
│   └── ...
└── logo.PNG
```

## Formats supportés :
- JPG/JPEG (recommandé)
- PNG
- WebP

## Tailles recommandées :
- Hero : 1920x1080px
- Galerie : 800x600px
- À propos : 400x400px
- Logo : 200x80px
